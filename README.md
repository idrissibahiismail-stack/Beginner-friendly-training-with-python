# Employee Salary Analysis (Python & Pandas)

## Project Overview
This project is a beginner-friendly **data analysis exercise** using Python and pandas.  
The goal is to practice **data manipulation, filtering, and aggregation** on an employee dataset.

The analysis focuses on employee salaries, departments, and years of experience, similar to basic HR or business analytics tasks.

---

## Dataset Description
The dataset is a **synthetic employee dataset** created for learning purposes.

**Columns:**
- `EmployeeID` – Unique identifier for each employee
- `Name` – Employee name
- `Department` – Department name (HR, Engineering, Marketing, Finance)
- `Salary` – Annual salary
- `YearsOfExperience` – Number of years of professional experience

---

## Objectives
- Create a DataFrame using pandas
- Explore and inspect tabular data
- Filter employees based on salary and experience
- Practice basic analytical thinking using Python

## Tools & Technologies

## Key Tasks Performed
- Created a pandas DataFrame from a dictionary
- Displayed and explored the dataset
- Filtered employees based on salary conditions
- Applied basic pandas operations (`query`, filtering, indexing)

## Example Analysis Questions
- Which employees earn less than a specific salary threshold?
- How many employees work in each department?
- How does experience relate to salary?
